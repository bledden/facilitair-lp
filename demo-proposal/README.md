# Facilitair Demo Proposal

This is a proposal demo site inspired by FutureHouse and Thinking Machines, maintaining Facilitair's brand identity (colors, fonts, animations) while adopting a cleaner, more research-focused aesthetic.

## Design Inspiration

- **FutureHouse**: Clean, minimal design with research-focused content
- **Thinking Machines**: Simple, functional layout emphasizing core features
- **Facilitair Brand**: Maintains teal (#5CE1E6), black (#100F0D), white (#FAFAFA), and Montserrat font

## Features

- Clean, minimal hero section
- Research-focused content sections
- Feature highlights in simple grid layout
- Stats section showcasing key metrics
- Supported models showcase
- About section
- Email signup CTA
- Responsive design
- Smooth scroll animations
- Maintains Facilitair's background animations and logo effects

## Local Development

Simply open `index.html` in a web browser. All assets are referenced relative to the demo-proposal directory.

## File Structure

```
demo-proposal/
├── index.html      # Main HTML file
├── styles.css      # All styles (maintains Facilitair brand)
├── script.js       # Interactive functionality
└── README.md       # This file
```

## Notes

- Logo path assumes `../logo.png` exists in parent directory
- No external dependencies except Google Fonts
- All animations and colors match Facilitair brand guidelines
- Does NOT reference pipedream, metorial, edison, or tinker
- Reflects beta.facilitair.ai offerings
